# 知识图谱可视化优化 设计文档

**日期**: 2026-05-26
**状态**: 已确认

## 1. 问题描述

当前知识图谱 47 个节点全部用简单圆形布局渲染，无视父子层级关系，导致：
- 节点密集重叠，标签互相遮挡
- 无缩放平移，无法浏览大图
- 无搜索筛选，无法快速定位
- 列表平铺展开，无分类结构

## 2. 目标

知识图谱同时支持：整体知识结构概览、快速搜索定位、识别薄弱环节。

## 3. 后端 API 优化

### 3.1 `/api/graph` 增强

新增查询参数：

| 参数 | 类型 | 说明 |
|------|------|------|
| `search` | string | 模糊匹配 topic 名称 |
| `parent` | string | 按 parent_topic 筛选，"" 或省略表示全部 |
| `min_level` | int | 最低熟练度阈值，默认 0 |

响应新增 `groups` 字段：

```json
{
  "nodes": [
    {"id": 1, "label": "Python 装饰器", "group": "Python 生态", "proficiency": 3}
  ],
  "edges": [{"from": 5, "to": 1}],
  "groups": ["Python 生态", "人工智能", "版本管理"]
}
```

### 3.2 修复幽灵节点

当前若 `parent_topic` 不在 `user_knowledge_graph` 表中，会用负数 ID 补一个虚拟节点。shrink 虚拟节点，改为直接用 `parent_topic` 字符串作为 group 标签，不生成独立的虚拟节点。

## 4. 前端 D3.js 力导向图

### 4.1 引入 D3.js v7

通过 CDN 引入，零构建依赖：
```html
<script src="https://d3js.org/d3.v7.min.js"></script>
```

### 4.2 力模拟配置

- `d3.forceLink(edges).id(d => d.id)` — 边约束，有父子关系的节点相互吸引
- `d3.forceManyBody().strength(-300)` — 节点互斥
- `d3.forceCenter(w/2, h/2)` — 居中
- `d3.forceCollide(20)` — 碰撞检测，防止重叠

启动动画：节点从中心扩散，持续约 1 秒。

### 4.3 视觉编码

- **节点大小**：`r = 8 + proficiency * 5`（熟练度 1→13px, 5→33px）
- **节点颜色**：按 `group`（parent_topic）分配，使用 `d3.schemeCategory10` 颜色方案
- **节点形状**：圆形 SVG circle
- **边**：灰色细线，有向（父→子），透明度 0.5
- **标签**：默认隐藏，hover 时显示（白色文字 + 暗色背景 tooltip）
- **画布**：SVG 替代 Canvas，尺寸适应容器宽度，高度 500px

### 4.4 交互

- **缩放平移**：`d3.zoom()`，滚轮缩放 + 鼠标拖拽平移
- **节点拖拽**：力模拟支持拖拽释放
- **Hover tooltip**：显示知识点名、熟练度、分类，定位在节点上方
- **点击节点**：高亮该节点及其直接关联节点，其余半透明

### 4.5 图例

图谱下方或右侧显示颜色-分类对应关系，可点击切换该分类的显隐。

## 5. 筛选与控制栏

图谱上方控制栏：

- **搜索框**：输入时实时过滤，匹配节点高亮，其余半透明（opacity 0.15）
- **分类下拉**：按 group 筛选，选项从 `groups` 字段动态生成
- **熟练度滑块**：range 0-5，隐藏低于阈值的节点
- **重置按钮**：恢复初始全量展示

## 6. 列表优化

将当前平铺列表改为按分类折叠：

- 每个 `parent_topic` 一组，默认展开
- 组头显示分类名 + 该分类下的节点数
- 点击组头折叠/展开
- 保留熟练度进度条
- 搜索和筛选条件同时影响图谱和列表

## 7. 文件变更清单

| 文件 | 改动 |
|------|------|
| `src/web/routes.py` | `/api/graph` 增强：查询参数、group 字段、去幽灵节点 |
| `src/web/templates/index.html` | 替换 Canvas 为 D3.js SVG；新增控制栏、图例、折叠列表 |

## 8. 验收标准

1. 图谱使用 D3 力导向布局渲染，节点不重叠
2. 支持滚轮缩放和拖拽平移
3. 搜索框输入关键词，匹配节点高亮
4. 分类下拉可筛选，熟练度滑块可过滤
5. 下方列表按分类折叠，搜索/筛选联动
6. `/api/graph` 接口支持 `?search=&parent=&min_level=` 参数
